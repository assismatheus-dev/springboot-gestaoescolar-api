# School Management API

API REST desenvolvida com Spring Boot para gerenciamento escolar, permitindo o controle de alunos, professores e turmas.

---

## Tecnologias utilizadas

- Java 22
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL
- Maven
- Swagger (Springdoc OpenAPI)

---

## Funcionalidades

- ✅ Cadastro de alunos  
- ✅ Cadastro de professores  
- ✅ Criação de turmas  
- ✅ Associação de alunos às turmas  
- ✅ Associação de professores às turmas  

---

## Banco de dados

- PostgreSQL  
- URL: `jdbc:postgresql://localhost:5432/spring_p2`
